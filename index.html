Machine Learning for Breast Cancer Screening

Data Overview

Machine learning is an exciting dimension of artificial intelligence that can revolutionize data analytics. Applying machine learning to health data may provide insights that can be used to increase efficiency, improve health outcomes, and save lives. This dashboard examines the results of applying various well-known machine learning models to breast cancer data from University of Wisconsin, Clinical Sciences Center to determine which model was most successful predicting malignant and benign diagnoses. First the models were applied to the total dataset. After using techniques to identify the most important variables, the models were then applied to those subqueries (Factor Analysis Variables, and Random Forest Variables) to see if the score or accuracy of the model was affected.

Key Takeaways

  • The most successful model was the Logistic Regression model applied to the total dataset.
  • The Nearest Neighbors model was consistent across all queries.
  • The highest grid search score was obtained using the total dataset.
  • Normal neural networks drastically outperformed deep neural networks with the highest score obtained using the total dataset.

Limitations

  • The dataset used to train the models was small, consisting of 569 rows of data.
  • Time; we would have liked to test and compare more queries
